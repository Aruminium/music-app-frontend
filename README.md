### CDのリリース日を知らせてくれるスマホアプリ

使用技術: React Native + expo

### install

1. ```$ npm install -g expo-cli```

### run & build

1. ```$ expo start```

2. ブラウザでlocalhost:19002

3. エミュレータを起動(Android)

### うまく動かない時

expoのキャッシュを消してみよう

```shell
expo r -c
```
